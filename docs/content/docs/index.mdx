# Introduction

Scripting Nodes is a visual scripting add-on for Blender 4.3+ that lets you create powerful Blender add-ons without writing Python code. Build your logic with nodes, and Scripting Nodes generates clean, production-ready code automatically.

## Why Scripting Nodes?

- **No Coding Required**: Create fully functional add-ons using an intuitive node-based interface
- **Learn by Doing**: See the generated Python code and learn Blender's API naturally
- **Rapid Prototyping**: Quickly test ideas and iterate on your add-on designs
- **Production Ready**: Export polished add-ons ready for distribution

## What Can You Build?

With Scripting Nodes, you can create:

- Custom operators and tools
- Panels and UI elements
- Property groups and settings
- Handlers and automation scripts
- Complete add-ons with menus and preferences
