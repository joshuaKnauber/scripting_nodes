# Math Nodes

Math nodes perform calculations on numeric values.

## Basic Operations

### Math
Performs basic arithmetic operations.

**Operations:**
- Add (+)
- Subtract (-)
- Multiply (ร)
- Divide (รท)
- Power (^)
- Modulo (%)

### Negate
Returns the negative of a value.

### Absolute
Returns the absolute (positive) value.

## Advanced Math

### Trigonometry
Sine, Cosine, Tangent and their inverses.

### Logarithm
Natural log, log base 10, and custom base logarithms.

### Square Root
Calculates the square root of a value.

### Clamp
Restricts a value to a minimum and maximum range.

### Map Range
Remaps a value from one range to another.

## Vector Math

### Vector
Creates a vector from X, Y, Z components.

### Separate Vector
Splits a vector into its X, Y, Z components.

### Vector Math
Operations on vectors: Add, Subtract, Dot Product, Cross Product, Normalize, Length.

### Distance
Calculates the distance between two points.

## Random

### Random
Generates a random value.

**Options:**
- Float (0.0 to 1.0)
- Integer (within range)
- Boolean (true/false)

### Random Seed
Sets the random seed for reproducible results.
