# Getting Started

This guide will help you install Scripting Nodes and create your first visual script.

## Installation

1. Download the latest release from the [GitHub releases page](https://github.com/joshuaknauber/scripting_nodes/releases)
2. In Blender, go to **Edit → Preferences → Add-ons**
3. Click **Install** and select the downloaded `.zip` file
4. Enable the add-on by checking the checkbox next to "Scripting Nodes"

## Creating Your First Node Tree

1. Open the **Scripting Nodes** editor from the editor type dropdown
2. Click **New** to create a new node tree
3. Press **Shift+A** to add nodes
4. Connect nodes together to build your logic
5. Your add-on code is generated automatically!

## Development Settings

In the Scripting Nodes panel (N-panel in the editor), you'll find development options:

- **Show Node Code**: Display generated code for individual nodes
- **Log Tree Rebuilds**: Log when node trees are regenerated
- **Force Production**: Generate production-ready code
